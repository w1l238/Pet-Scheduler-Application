# Admin Dashboard

The Admin Dashboard provides administrators with a comprehensive set of tools to manage every aspect of the pet scheduling business.

## Features

### Central Calendar
Admins have a centralized calendar view that displays all appointments from all clients, providing a complete overview of the business's schedule.

### Appointment Management
Admins can manage all client appointments. This includes:
*   Viewing all appointments filterable by status.
*   Approving or denying pending appointment requests.
*   Marking scheduled appointments as "Completed" which automatically generates an invoice.

### Client Management
Admins have full control over client records:
*   View a list of all registered clients and search by name.
*   Edit client information, including their name, contact details, and role (Client or Admin).
*   Delete client accounts.

### Pet Management
Admins can view and manage all pets registered in the system, including viewing their profiles and deleting records if necessary.

### Service Management
This section allows admins to define the services offered by the business:
*   Add new services with a name, description, and price.
*   Edit existing service details.
*   View statistics on how many times each service has been booked.
*   Delete services.

### Invoice Management
Admins can track and manage all financial records:
*   View a list of all invoices, searchable by client or pet name.
*   See summary cards for "Total Collected" and "Total Outstanding" revenue.
*   Mark outstanding invoices as "Paid".
*   Link directly from an invoice to its corresponding appointment details.

### Profile & Settings
*   **Admin Profile:** Admins can manage their own user profile.
*   **Business Settings:** Admins can configure application-wide settings, such as booking rules and email templates.
