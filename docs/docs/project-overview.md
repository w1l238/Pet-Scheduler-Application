# Project Overview

Welcome to the documentation for the Pet Scheduler Application!

This application is a web-based information system designed to help small pet grooming or scheduling businesses manage their clients, pets, appointments, services, and invoices efficiently.

## Key Features

*   **Client Management:** Add, view, edit, and delete client information.
*   **Pet Management:** Associate multiple pets with each client, including pet details and profile photos.
*   **Appointment Scheduling:** Schedule, view, approve, and cancel appointments.
*   **Service Management:** Define and manage various services offered by the business.
*   **Invoice Management:** Generate and track invoices for completed appointments.
*   **User Authentication & Authorization:** Secure login for both administrators and clients, with role-based access control.
*   **Automated Reminders:** Send email reminders for upcoming appointments.
*   **Notifications:** Real-time notifications for admins (pending appointments) and clients (appointment status changes).
*   **Profile Pages:** Dedicated profile pages for clients and pets with their respective details.

## Technology Stack

The application is built using the following technologies:

*   **Frontend:** React
*   **Backend:** Node.js with Express
*   **Database:** PostgreSQL
